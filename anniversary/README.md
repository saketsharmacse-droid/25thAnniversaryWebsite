# 💖 25th Wedding Anniversary Website

A heartfelt and simple web project created to celebrate my parents' **25th wedding anniversary**.  
This website is designed to preserve memories, showcase beautiful moments, and create a digital celebration experience.

---

## 🌸 About the Project

This is a lightweight, UI-focused website built using **HTML, CSS, and JavaScript** with minimal backend support.  
The main goal of this project is to present a clean, emotional, and visually pleasant experience that highlights special memories, photos, and milestones of the journey of love and togetherness.

---

## 🛠️ Tech Stack

- HTML5 – Structure of the website  
- CSS3 – Styling and responsive design  
- JavaScript – Interactivity and dynamic elements  
- Minimal Backend – For handling basic data or photo links  

---

## 💝 Features

- Beautiful anniversary-themed UI  
- Photo/memory gallery  
- Personalized messages and timeline of memories  
- Responsive design for mobile and desktop  
- Simple and fast-loading interface  

---

## 🎯 Purpose

This project is not commercial.  
It is a personal gift and tribute to my parents for their **25 years of togetherness, love, and memories**.
